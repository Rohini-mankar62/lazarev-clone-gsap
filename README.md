# Animated Website (Frontend Project)

This is a frontend project where I created an interactive website using modern animations and smooth scrolling.

## Features
- GSAP animations
- ScrollTrigger effects
- Locomotive Scroll integration
- Responsive layout
- Smooth UI interactions

## What I Learned
- How to use GSAP for animations
- Scroll-based animation techniques
- Debugging animation issues
- Improving UI/UX with micro-interactions

## Note
This project is inspired by modern design agency websites and was built for learning purposes.

## Tech Used
- HTML
- CSS
- JavaScript
- GSAP
- Locomotive Scroll
